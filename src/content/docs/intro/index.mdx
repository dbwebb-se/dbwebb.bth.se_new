Kursen Webbteknologier, a.k.a. webtec, lär ut grunderna för webbteknologier som HTML, CSS och JavaScript och hur de samverkar i webbläsaren för att skapa webbplatser och webbapplikationer.   

Detta är en introduktionskurs för den som vill lära sig teknikerna från grunden. Kursen kräver inga förkunskaper och hanterar helheten kring en webbapplikation. Till att börja med fokuserar vi på HTML och CSS. Därefter lär vi oss hur programmeringsspråket JavaScript fungerar i webbläsaren tillsammans med HTMl och CSS.

HTML och CSS ger grundförutsättningarna för hur en webbapplikation kan byggas. Genom att använda JavaScript på klientsidan så kan webbapplikationen bli mer dynamisk.

Sammantaget blir kursen en introduktion och orientering i de grundtekniker som används på klientsidan (webbläsaren) för att skapa webbplatser.




Innehåll
------------------------

Kursen omfattar följande områden:

* HTML element, dess uppbyggnad och användande. 

* CSS konstruktioner, dess uppbyggnad och användande. 

* Webbplatsens innehåll med text, bilder, ljud, video, typografi. 

* Layout av webbplats med positionering, float och flexbox. 

* Responsiv webbplats med media queries. 

* Programmering med JavaScript i webbläsaren med variabler, villkor, loopar och funktioner samt moduler inklusive datastrukturer som array och objekt. 

* Webbläsaren som programmeringsmiljö med “Document Object Model” (DOM) och event-driven programmering. 

* Introduktion till webbaserade API:er såsom till exempel HTML Web Storage API. 

* Verktyg och tekniker som används vid utveckling av webbaserade applikationer med HTML, CSS och JavaScript med webbläsaren som programmeringsmiljö.
